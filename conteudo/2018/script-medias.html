Script Médias

Nestes momentos a minha sinceridade não deixa de agir. Por que você precisaria de um script pra calcular notas? hehehe. Ok, ele além de calcular notas, registra nome do professor, série, nome do aluno, pontos adicionais, faltas e observações e armazena esses dados num documento de texto. Então se você é um professor, que por algum motivo quer guardar as lindas notas de seus alunos, esse script é pra você. Sim, se talvez você só queira o script para fins de adquirir conhecimento, o código está todo comentado.

Como executar?

O Script é feito em python3 roda em linux e mac normalmente sendo preciso apenas ir, pelo terminal, até o pasta onde está o script e executar pelo terminal da seguinte forma:

$ python3 nome_do_script.py

Em Windows é necessario ter o python instalado no seu computador. Para isso precisamos ir ao site do Python e baixar a versão mais recente, para não haver erros. Feito isso a instalação é bem simples. Ok, instalou? se sim, vá até a pasta do script e execute-o com double click. Se tiver problemas, pesquise.
