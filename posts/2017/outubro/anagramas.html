Permutação + Anagramas

A ideia surgiu aleatoriamente, enquanto o professor de matemática resolvia alguns exercicios de Análise Combinatória. A permutação com a palavra "periquito" deu exatamente 181.440, ele, ironicamente, disse que quem quisesse fazer o anagrama inteiro em casa podia, nesse momento eu até perguntei alto "E se eu fizesse?". Tranquilo, fui para casa, à noite comecei a trabalhar nesse projeto que a princípio parecia impossível, várias e várias tentativas de encontrar uma lógica que poderia ser aplicada, mas, só falhas. Só encontrei uma solução, quando fui tomar aquele banho de 2 minutos, ate aí ja tinha passado algumas horas na frente do PC. Terminei o código, se me recordo bem, nessa mesma noite, nos outros dias só fui fazer o trabalho de comentar todo o código que é muito chato, mas, acho interessante. Falei toda história mas não falei o que ele faz, bom, ele simplesmente calcular o número de anagramas, mostra todos os anagramas, caso o usuário queira, e cria um documento de texto com esses anagramas. E interessante salientar que para o documento de texto ser totalmente escrito, é interassante que você espere todos os anagramas aparecerem, por algum motivo mesmo os anagramas estando em 80 mil caso você cancele o script o documento de texto não vai parar exatamente onde você parou. Sinta-se avontade para comentar, especialmente sobre bugs. Obrigado.

Como usar?

O Script é feito em python3 roda em linux e mac normalmente sendo preciso apenas ir, pelo terminal, até o pasta onde está o script e executar pelo terminal(de preferência como root) da seguinte forma:

# python3 nomedoscript.py

Em Windows é necessario ter o python instalado no seu computador. Para isso precisamos ir ao site do Python e baixar a versão mais recente, para não haver erros. Feito isso a instalação é bem simples. Ok, instalou? se sim, vá até a pasta do script e execute-o com double click. Se tiver problemas, pesquise.
